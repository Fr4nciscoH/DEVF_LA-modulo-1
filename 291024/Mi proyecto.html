Listas en lineas de código

Para insertar listas ordenadas en un documento html es necesario usar la etiqueta <ol> y <li>

Por ejemplo:

  1. Esta
  2. es
  3. una
  4. lista
  5. ordenada.
Se considera ordenada porque está numerada o con letras

Ahora para insertar una lista no ordenada (solo lleva viñetas) la etiqueta <ol> se cambia por <ul>

Por ejemplo:
  • Esta
  • es
  • una
  • lista
  • no
  • ordenada.
